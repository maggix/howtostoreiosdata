“How should I store data in my iOS app securely?”

iOS has many excellent features to protect your user's data. However, finding your way through all the slightly different APIs and settings for all the different storage methods can be challenging.

That's why I built howtostoreiosdata.com. Run through a few basic questions, and get an advice on how to store your data as securely as possible, along with a code sample.

Who built this?

I'm Erik Romijn, App Maker from Amsterdam, The Netherlands. I love making apps - for both mobile and web. I do a mix of freelance work and independent projects, mostly with Django and iOS.

I live in the wonderful city of Appsterdam, where we are working to create the best place in the world to be or become an App Maker.

Why was this built?

I was looking into secure iOS data storage, and found that it can be very difficult to discover how particular data is best protected, and what the API looks like. It's important that security is as simple as possible, so I built this site to improve that.

Can I help?

Yes! This is a work in progress, and there is much left to do. All the code is on GitHub. Have a look through the open issues. And even if you can't contribute code, you can suggest improvements by creating a new issue.
